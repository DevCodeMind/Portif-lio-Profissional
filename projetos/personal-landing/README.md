# Personal Trainer Landing Page

Página de destino moderna e responsiva para personal trainer, com design energizante e motivacional focado em resultados.

## 🎯 Características

- **Design Moderno**: Interface clean com elementos visuais energizantes
- **Responsivo**: Otimizado para desktop, tablet e mobile
- **Animações**: Transições suaves e efeitos de scroll
- **CTA Forte**: Chamadas para ação estratégicas via WhatsApp
- **SEO Otimizado**: Meta tags e estrutura semântica

## 🚀 Tecnologias

- HTML5 semântico
- CSS3 com variáveis CSS e Grid/Flexbox
- JavaScript vanilla (Intersection Observer)
- Google Fonts (Inter + Bebas Neue)
- Imagens otimizadas do Unsplash

## 📱 Seções

1. **Header Fixo**: Navegação com glassmorphism
2. **Hero**: Apresentação com estatísticas e CTAs
3. **Serviços**: Grid de serviços oferecidos
4. **Sobre**: Experiência e credenciais
5. **CTA**: Chamada final para ação
6. **Footer**: Informações de copyright

## 🎨 Paleta de Cores

- **Primária**: Laranja (#f59e0b) - Energia e motivação
- **Background**: Preto (#0a0a0a) - Sofisticação
- **Cards**: Cinza escuro (#1a1a1a) - Contraste moderno

## 📞 Contato Integrado

- WhatsApp direto para agendamento
- Links otimizados com mensagens pré-definidas

## 🔧 Personalização

Para personalizar:
1. Substitua "Nome" pelo nome real do personal trainer
2. Atualize as estatísticas na seção hero
3. Modifique os serviços oferecidos
4. Altere as imagens para fotos reais
5. Atualize o número do WhatsApp

## 📊 Performance

- Carregamento rápido (HTML inline)
- Imagens otimizadas
- CSS eficiente
- JavaScript leve
