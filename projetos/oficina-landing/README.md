# Oficina Mecânica Landing Page

Página de destino profissional e confiável para oficina mecânica, com design robusto focado em confiança e qualidade técnica.

## 🔧 Características

- **Design Profissional**: Interface robusta com tons azuis e cinzas
- **Responsivo**: Otimizado para desktop, tablet e mobile
- **Animações**: Transições suaves e efeitos de scroll
- **CTA Forte**: Chamadas para orçamento via WhatsApp
- **SEO Otimizado**: Meta tags e estrutura semântica

## 🚀 Tecnologias

- HTML5 semântico
- CSS3 com variáveis CSS e Grid/Flexbox
- JavaScript vanilla (Intersection Observer)
- Google Fonts (Inter + Bebas Neue)
- Imagens otimizadas do Unsplash

## 📱 Seções

1. **Header Fixo**: Navegação profissional com glassmorphism
2. **Hero**: Apresentação com diferenciais e CTAs
3. **Serviços**: Grid detalhado de serviços mecânicos
4. **Sobre**: Experiência e credenciais
5. **CTA**: Chamada final para orçamento
6. **Footer**: Informações de copyright

## 🎨 Paleta de Cores

- **Primária**: Azul (#2563eb) - Confiança e profissionalismo
- **Background**: Cinza claro (#f8fafc) - Limpeza e modernidade
- **Cards**: Branco (#ffffff) - Transparência
- **Acentos**: Verde (#10b981) - Sucesso/qualidade

## 📞 Contato Integrado

- WhatsApp direto para orçamento
- Links otimizados com mensagens pré-definidas

## 🔧 Personalização

Para personalizar:
1. Substitua "Nome" pelo nome real da oficina
2. Atualize os serviços oferecidos
3. Modifique as imagens para fotos reais da oficina
4. Atualize o número do WhatsApp
5. Ajuste as estatísticas na seção sobre

## 📊 Performance

- Carregamento rápido (HTML inline)
- Imagens otimizadas
- CSS eficiente
- JavaScript leve

## 🛠️ Serviços Incluídos

- Diagnóstico Eletrônico (scanner digital, códigos de erro)
- Manutenção Preventiva (óleo, filtros, alinhamento)
- Reparos Mecânicos (motor, freios, suspensão)
- Sistema Elétrico (bateria, alternador, ignição)
- Funilaria e Pintura (lataria, pintura, polimento)
- Inspeção Veicular (segurança, emissões)
