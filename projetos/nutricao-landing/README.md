# Landing — Nutrição

Landing para **nutrição** — **modelo split-screen**: hero em duas metades (gradiente verde + painel branco com lista e CRN), faixas alternadas claras/creme com três colunas, CTA escuro no rodapé — *Lora* + *Plus Jakarta Sans* (sem grid bento).

## Como visualizar

Abra o arquivo `index.html` no navegador (duplo clique).

## O que personalizar

- **Nome** no header e rodapé (`Dra. Nome Sobrenome`)
- **CRN** nos badges e no rodapé (formato de exemplo: `CRN 0 00000`)
- Texto “Sobre mim” com `[faculdade]` e focos entre colchetes
- **WhatsApp** na faixa de contato final
- **E-mail** no rodapé
