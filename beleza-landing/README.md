# Salão de Beleza Landing Page

Página de destino elegante e sofisticada para salão de beleza, com design feminino e acolhedor focado em bem-estar e transformação.

## 🎨 Características

- **Design Elegante**: Interface sofisticada com tons rosados e dourados
- **Responsivo**: Otimizado para desktop, tablet e mobile
- **Animações**: Transições suaves e efeitos de scroll
- **CTA Forte**: Chamadas para ação estratégicas via WhatsApp
- **SEO Otimizado**: Meta tags e estrutura semântica

## 🚀 Tecnologias

- HTML5 semântico
- CSS3 com variáveis CSS e Grid/Flexbox
- JavaScript vanilla (Intersection Observer)
- Google Fonts (Inter + Playfair Display)
- Imagens otimizadas do Unsplash

## 📱 Seções

1. **Header Fixo**: Navegação elegante com glassmorphism
2. **Hero**: Apresentação com diferenciais e CTAs
3. **Serviços**: Grid de serviços com preços
4. **Sobre**: Experiência e estatísticas
5. **CTA**: Chamada final para agendamento
6. **Footer**: Informações de copyright

## 🎨 Paleta de Cores

- **Primária**: Rosa (#ec4899) - Feminilidade e energia
- **Secundária**: Dourado (#d4af37) - Luxo e sofisticação
- **Background**: Creme (#faf7f5) - Calor e acolhimento
- **Cards**: Branco (#ffffff) - Pureza e limpeza

## 📞 Contato Integrado

- WhatsApp direto para agendamento
- Links otimizados com mensagens pré-definidas

## 🔧 Personalização

Para personalizar:
1. Substitua "Nome" pelo nome real do salão
2. Atualize os preços dos serviços
3. Modifique os serviços oferecidos
4. Altere as imagens para fotos reais do salão
5. Atualize o número do WhatsApp
6. Ajuste as estatísticas na seção sobre

## 📊 Performance

- Carregamento rápido (HTML inline)
- Imagens otimizadas
- CSS eficiente
- JavaScript leve
