# Barbearia Landing Page

Página de destino clássica e elegante para barbearia, com design masculino e sofisticado focado em tradição e qualidade.

## ✂️ Características

- **Design Clássico**: Interface elegante com tons dourados e marrons
- **Responsivo**: Otimizado para desktop, tablet e mobile
- **Animações**: Transições suaves e efeitos de scroll
- **CTA Forte**: Chamadas para agendamento via WhatsApp
- **SEO Otimizado**: Meta tags e estrutura semântica

## 🚀 Tecnologias

- HTML5 semântico
- CSS3 com variáveis CSS e Grid/Flexbox
- JavaScript vanilla (Intersection Observer)
- Google Fonts (Inter + Cinzel)
- Imagens otimizadas do Unsplash

## 📱 Seções

1. **Header Fixo**: Navegação elegante com glassmorphism
2. **Hero**: Apresentação com diferenciais e CTAs
3. **Serviços**: Grid de serviços com preços
4. **Sobre**: Experiência e estatísticas
5. **CTA**: Chamada final para agendamento
6. **Footer**: Informações de copyright

## 🎨 Paleta de Cores

- **Primária**: Dourado (#d4af37) - Luxo e tradição
- **Secundária**: Marrom (#8b4513) - Clássico e robusto
- **Background**: Preto (#1a1a1a) - Sofisticação masculina
- **Cards**: Cinza escuro (#2a2a2a) - Contraste elegante

## 📞 Contato Integrado

- WhatsApp direto para agendamento
- Links otimizados com mensagens pré-definidas

## ✂️ Personalização

Para personalizar:
1. Substitua "Nome" pelo nome real da barbearia
2. Atualize os preços dos serviços
3. Modifique os serviços oferecidos
4. Altere as imagens para fotos reais da barbearia
5. Atualize o número do WhatsApp
6. Ajuste as estatísticas na seção sobre

## 📊 Performance

- Carregamento rápido (HTML inline)
- Imagens otimizadas
- CSS eficiente
- JavaScript leve

## 💇‍♂️ Serviços Incluídos

- Cortes de Cabelo (modernos, clássicos, degradê)
- Barba Completa (modelagem, aparagem, hidratação)
- Combo Completo (corte + barba + sobrancelhas)
- Tratamentos Capilares (hidratação, reconstrução)
- Limpeza de Pele (profunda, hidratação facial)
